    Alarm Clock

    Summary:

    This project employs Python, the Spotify API, and various Python libraries to develop an alarm clock application capable of utilizing MP3 files as the audio source for alarm activation. The inspiration for this project arose from a personal experience when I was awakened by a jarring alarm sound, compounded by the inconvenience of having to purchase songs for $5 each. The objective of this application is to address this issue by enabling users to wake up to their preferred songs, enhancing their morning experience.

    Comix

    Summary:

    The Java project is a comprehensive comic book collection management system designed to efficiently import, organize, and manipulate user's comic collections. Leveraging a database or .csv file, users can seamlessly add, delete, and edit comic entries, ensuring a dynamic and personalized collection. The system offers a robust search functionality, allowing users to quickly locate specific comics within their database. Additionally, users can export their entire collection in various formats such as .XML, .json, and .csv files, ensuring data portability. This project not only simplifies the management of comic collections but also enhances user experience by providing versatile tools for organization and data manipulation. Whether users are avid collectors or occasional enthusiasts, this Java-based system offers a user-friendly interface and powerful features to streamline the management of their comic book treasures.
